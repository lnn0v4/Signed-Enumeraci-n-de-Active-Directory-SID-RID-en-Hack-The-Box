# Signed-Enumeraci-n-de-Active-Directory-SID-RID-en-Hack-The-Box
Script en Python3 desarrollado para la máquina Signed de Hack The Box, que permite enumerar objetos de Active Directory vía MSSQL, extrayendo SID y RID e identificando si corresponden a Users, Groups o Computers, con un enfoque simple y educativo.
